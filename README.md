<h2>Prerequisites</h2>
Make sure you have Git & node installed on your system.
Verify this by running this on your system : 
```console
git --version
```
```console
node --version
npm --version
```

<h2>Installation</h2>

```console
git clone https://github.com/yobslob/Binary-Growth.git
```

```console
cd Binary-Growth
```

```console
npm install
```

<h2>Local Development</h2>

```console
npm run dev
```
